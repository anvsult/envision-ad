# Envision Ad

Envision Ad is a platform that connects media owners with advertisers, made for Visual Impact.


## Description

Visual Impact is a startup with the goal of providing affordable advertisement opportunities to small businesses. 
To help with that mission, Envision Ad is an online platform that allows Media Owners (businesses and individuals who 
own digital screens or physical posters in public spaces) to list their ad spaces so that advertisers can publish
their ads on them.
